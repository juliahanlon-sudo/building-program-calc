# Salesforce Space Planning Calculator

A React + Vite web app for building program / space planning, with Firebase
Authentication, Firestore-backed scenario saving/sharing, invite-based signup,
and an admin panel.

## Running locally

Requires Node.js 18+ (tested on Node 24).

```bash
npm install      # install dependencies (first time only)
npm run dev      # start the dev server → http://localhost:5173
```

Then open **http://localhost:5173** in your browser.

## Other commands

```bash
npm run build    # production build → dist/
npm run preview  # serve the production build locally
npm run deploy   # build + deploy to Firebase Hosting (requires firebase login)
```

## Project structure

```
index.html            App entry HTML
vite.config.js        Vite + React config
firebase.json         Firebase Hosting config (deploys the dist/ folder)
.firebaserc           Firebase project alias (building-program-calc)
src/
  main.jsx            React entry; handles auth gate (login vs app)
  firebase.js         Firebase SDK init + config
  firebaseHelpers.js  Auth, invites, users, scenario CRUD helpers
  App.jsx             Main calculator UI (the bulk of the app)
  LoginScreen.jsx     Sign in / sign up / password reset
  ScenarioManager.jsx Save / load / share scenarios
  AdminPanel.jsx      User & invite management (admins only)
```

## Notes

- **Firebase backend:** the app talks to the live Firebase project
  `building-program-calc` (config in `src/firebase.js`). The web API key there is
  a public client identifier (safe to commit); access is governed by Firestore
  security rules in the Firebase console, not by hiding the key.
- **Signup is invite-only:** new users need an invite link generated by an admin
  in the Admin Panel.
